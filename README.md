# 🤖 AI-CHATBOT

A full-stack AI chatbot application built using React, Node.js, Express, LangChain, and Groq API.  
This project provides intelligent AI conversations with a modern and responsive user interface.

---

# 🚀 Features

- AI Chat Interface
- Real-time AI Responses
- React Frontend
- Express Backend
- LangChain Integration
- Groq API Integration
- Responsive Design
- Fast API Communication
- Clean UI

---

# 🛠️ Tech Stack

## Frontend
- React
- Vite
- Tailwind CSS
- Axios

## Backend
- Node.js
- Express.js
- LangChain
- Groq API
